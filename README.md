## Overview

This portfolio is designed to present real-world applications with a focus on product thinking, clean user experience, and maintainable system design.

Rather than listing projects casually, the goal is to highlight how each application is built, structured, and intended to scale.

## Featured Project

### Soul Link

A persona-based social platform that allows users to create and manage multiple identities, each with its own feed, conversations, and connections.

- Multi-persona identity system
- Persona-scoped data architecture
- Full-stack implementation with authentication and state management
- Designed for intentional interaction and identity separation
